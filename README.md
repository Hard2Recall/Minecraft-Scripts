# This is a minescript folder with 2 of my scripts.

SETING UP:  
in `config.txt` u put your file path to `python.exe`
you CANT use this whole folder you can copy the `getinfo.py` and `kill.py` files into your minescript folder that is loocated in .minecraft folder.

USAGE:  
EVERYTHING IS RUN IN GAME.  
`getinfo.py` - in game chat use \getinfo command and text is gonna pop in chat, you press "[" key to get info about an entity and/or block your crosshair is placed on.  
Press "]" key to stop the script.

`kill.py` - in game chat use \kill command and script will automatically start and will press left clik (or your attack button) every time any entity is in corsshair and is closer than 4 blocks.   
Press "]" to stop the script from running.
